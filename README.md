# myjob
自动化抓取招聘信息，数据存储，数据分析。分析出有价值的信息，如技术趋势，招聘需求，行业类型，薪资待遇等。具体查看doc下的业务方案。

#开发进度-目前已完成的功能
1、通过抓取源url，以及过滤关键字，然后抓取数据，存储到redis数据库。
2、数据提取程序，从redis缓存中读出数据，然后提取相关字段存入mysql数据库。
